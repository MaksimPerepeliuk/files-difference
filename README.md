## Setup using npm

```sh
$ npm install -g generate-difference-cli
```

## Setup using git clone
clone this repository and npm install

```sh
$ npm install
```

* program supports input file types: `.yml` `.ini` `.json`
* and have three output types: `tree` `plain` `json`
